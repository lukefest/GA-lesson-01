Terminal & Git

Common Terminal Commands

  • cd - change directory
  • cd .. - up one level directory
  • ls - list directory
  • ls -lah - list directory detailed

Common Git Commands

  • git init - start new project
  • git status - show the status of files
  • git add - adds file (e.g. git add HTML/index.html)
  • git commit -m 'message here' - commit with message
  • git push - pushes all files to GitHub(?)

Javascript

Common JS Comparison Operators

  • > - more than
  • < - less than
  • >= - more than or equal to
  • <= less than or equal to
  • === equal to
  • !== not equal to
